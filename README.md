## Planned Features 

- Automatically generated monthly diaries that summarize media consumed, amount read / listened, events taken part in, achievements reached.. look back on your journey, like a diary
- Detailed statistics: On top of standard metrics (pages read, minutes listened, ..), group and filter by fine-grained aspects: active vs passive learning, with or without subtitles, reading only vs reading along with accompanying audio, ..
- Track book clubs
- ...
